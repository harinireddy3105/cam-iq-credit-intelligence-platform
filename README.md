# Loan Fraud Detection & Credit Intelligence Platform

## Description
This project is designed to help banks evaluate loan applications by verifying company authenticity and identifying potential fraud risks.

## Features
- Company document verification
- Fraud risk analysis
- Loan approval decision support
- Third-party evaluation system for banks
